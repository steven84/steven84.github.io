如何在windows平台上安装Scrapy

一、Scrapy简介

官方主页:http://www.scrapy.org/

二、Python

Python2.7版本 64位系统，不要选择Python3.0

三、遇到的问题

安装时提示缺少lxml、libxml2，通过搜索引擎找到编译好的64位windows版本，官方只有32位。

http://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml
http://www.lfd.uci.edu/~gohlke/pythonlibs/#libxml2

下载之后，使用

pip install libxml2_python-2.9.3-cp27-none-win_amd64.whl
pip install lxml-3.5.0-cp27-none-win_amd64.whl
